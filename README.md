## Installation

Clone the repository

```bash
  git clone https://github.com/vickyshaw29/cloud-pulse.git
```
Go the project directory and install the dependencies
```bash
  cd cloud-pulse
  npm install
```
Run the project locally(ensure Firebase setup)
```bash
  npm run dev
```
